# Развёртывание на Vercel

## Шаг 1: Подготовка токенов

### GitHub Personal Access Token
1. Зайди на https://github.com/settings/tokens
2. Нажми "Generate new token (classic)"
3. Дай название: `orchestration-deploy`
4. Выбери права: `repo` (полный доступ к репозиториям)
5. Нажми "Generate token"
6. **Скопируй токен** (больше не сможешь его увидеть!)

### Vercel Token
1. Зайди на https://vercel.com/account/tokens
2. Введи название: `orchestration`
3. Нажми "Create"
4. **Скопируй токен**

## Шаг 2: Запусти скрипт деплоя

Открой терминал и выполни:

```bash
cd /Users/alexeigrosul/Desktop/orchestration
bash deploy.sh
```

## Шаг 3: Следуй инструкциям

Скрипт попросит тебя ввести:
1. **GitHub Token** - вставь скопированный токен
2. **Vercel Token** - вставь скопированный токен
3. **GitHub username** - твоё имя пользователя на GitHub

## Шаг 4: Готово! 🎉

После выполнения скрипта ты получишь:
- ✅ Созданный GitHub репозиторий
- ✅ Загруженный код на GitHub
- ✅ Задеплоенное приложение на Vercel
- ✅ **Рабочую ссылку на приложение**

---

**Заметки:**
- Скрипт не отправляет твои токены никуда
- Токены используются локально на твоём компьютере
- После деплоя ты сможешь удалить токены из GitHub/Vercel если хочешь
